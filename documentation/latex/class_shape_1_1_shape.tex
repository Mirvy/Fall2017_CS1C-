\subsection{Detailed Description}
\doxyref{Shape}{p.}{class_shape_1_1_shape} A class hierarchy used to create 2d shapes which are drawn to a canvas.

\doxyref{Shape}{p.}{class_shape_1_1_shape} A base abstract class is used to provide a common interface for all inherited shapes.

Uses enum shape to differentiate between different derived shape classes.
